01

การเข้าสู่ระบบ

ใช้ MSU Google Account เพื่อเข้าสู่ระบบ

1
กดปุ่ม "Sign in with Google"
บนหน้าเข้าสู่ระบบ ให้กดปุ่ม Sign in with Google และเลือกบัญชี @msu.ac.th เท่านั้น
2
รอการอนุมัติ (ครั้งแรก)
หากเป็นการเข้าสู่ระบบครั้งแรก ต้องรอเจ้าหน้าที่อนุมัติบัญชีก่อน จึงจะเข้าใช้งานได้
3
เลือก Dashboard ตามบทบาท
ระบบจะพาไปยัง Dashboard ที่ตรงกับบทบาทของคุณโดยอัตโนมัติ (นิสิต / อาจารย์ / เจ้าหน้าที่)
⚠️ ระบบรองรับเฉพาะบัญชี @msu.ac.th เท่านั้น บัญชี Gmail ทั่วไปไม่สามารถเข้าใช้งานได้
02

การค้นหาวารสาร

ตรวจสอบวารสารผ่าน ISSN หรือ eISSN

1
ไปที่เมนู "ค้นหาวารสาร"
เลือกเมนูค้นหาวารสารจาก Sidebar ด้านซ้ายมือ
2
กรอกหมายเลข ISSN
กรอกหมายเลข ISSN ในรูปแบบ XXXX-XXXX หรือ XXXXXXXX (8 หลัก) แล้วกดค้นหา
3
เลือกวิธีดึงข้อมูล
Web Scraping — ไม่ต้องใช้ API Key ใช้เวลา 3–8 วินาที (แนะนำ)
API — เร็วกว่า แต่ต้องมี Scopus API Key และมี Rate Limit
Web Scraping (แนะนำ)
  • ✅ ไม่ต้องใช้ API Key
  • ✅ ไม่มี Rate Limit
  • ✅ ฟรี
  • ⏱ ใช้เวลา 3–8 วินาที
API
  • ⚡ เร็วกว่า (<1 วินาที)
  • ✅ ฟรี (สถาบัน)
  • 🔑 ต้องมี API Key
  • ⚠️ จำกัด 20,000 ครั้ง/สัปดาห์
03

การอ่านผลลัพธ์

ทำความเข้าใจสีและสถานะของวารสาร

ผ่านเกณฑ์สากล (Scopus)
วารสารอยู่ใน Scopus Q1–Q2 (ป.เอก) หรือ Q1–Q3 (ป.โท) สถานะ Active ไม่ติด Unwanted
ผ่านเกณฑ์มาตรฐาน (TCI กลุ่ม 1–2)
วารสารอยู่ใน TCI กลุ่ม 1 (ป.เอก) หรือ กลุ่ม 1–2 (ป.โท) และยังตีพิมพ์อยู่
แจ้งเตือน: MSU Unwanted แต่ Scopus Active
วารสารอยู่ใน Scopus แต่ปรากฏใน MSU Unwanted List ไม่สามารถใช้ยื่นได้
ปฏิเสธ: MSU Unwanted
วารสารอยู่ในรายการ MSU Unwanted Journals ห้ามนำไปยื่น Pre-T3 / T3 โดยเด็ดขาด
ปฏิเสธ: Predatory Journal (Watchlist)
วารสารปรากฏใน Beall's List หรือรายการวารสารล่อลวง ห้ามนำไปใช้โดยเด็ดขาด
04

การยื่นแบบฟอร์ม Pre-T3

ขั้นตอนการสร้างและยื่นเอกสาร Pre-T3

1
ค้นหาวารสารที่ผ่านเกณฑ์
ค้นหาวารสารที่ต้องการ และตรวจสอบว่าผ่านเกณฑ์สำหรับระดับการศึกษาของคุณ
2
กดปุ่ม "ยื่น Pre-T3"
บนหน้าผลลัพธ์ กดปุ่ม "ยื่น Pre-T3" ระบบจะนำข้อมูลวารสารกรอกให้อัตโนมัติ
3
กรอกข้อมูลเพิ่มเติมและยืนยัน
ตรวจสอบข้อมูลให้ถูกต้อง จากนั้นกดยืนยันส่งให้อาจารย์ที่ปรึกษาพิจารณา
4
ติดตามสถานะ
ตรวจสอบสถานะการพิจารณาได้ที่เมนู "สถานะ Pre-T3" ใน Sidebar
05

MSU Unwanted Journals

รายการวารสารที่ มมส. ไม่อนุมัติการใช้งาน

MSU Unwanted Journals คือรายการวารสารที่มหาวิทยาลัยมหาสารคามประกาศว่า ไม่อนุมัติให้นิสิตนำไปใช้เป็นหลักฐานประกอบการยื่นเอกสาร Pre-T3 หรือ T3 แม้ว่าวารสารนั้นจะอยู่ใน Scopus หรือ TCI ก็ตาม

ℹ️ รายการ MSU Unwanted ดูได้ที่เมนู "MSU Unwanted Journals" ใน Sidebar ซึ่งเจ้าหน้าที่จะอัปเดตอยู่เสมอ
